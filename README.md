# E-commerce AVANTI

Projeto de e-commerce moderno, responsivo e focado na experiência do usuário. Desenvolvido com HTML, CSS e JavaScript, simula a estrutura de uma loja virtual real.

Site 100% responsivo e funcional.

### Tecnologias usadas
- HTML
- CSS
- Javascript
### Bibliotecas usadas
- SwiperJs
